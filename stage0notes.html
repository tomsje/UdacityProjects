Overview of the internet

What happens when you open a web page?

A webpage is a text document written in HTML. When you enter a web address or URL in your web browser internet connects you to the webservers where the documents live. Your Computer connects to the internet using http to contact web servers. Http is the language that the internet communicates to and from your computer to the webserver. Introduction to HTML

What is a webpage?

A web page is a text document written in HTML. HTML stands for Hyper Text Markup Language. The text or content is displayed as specified by the “markup” or the formatting you see.
Content can be anything from text to images or links to other pages.

Computers ARE stupid beware!

Computers are completely literal and are easily confused if the syntax is incorrect. Pay close attention to syntax to avoid unexpected behavior. Remember that it will do exactly you say, so say it correctly!
Don't worry Programmers can’t remember everything if you forget something - look it up. It is very easy to lookup syntax on the internet. Try searching on Google.com.

HTML vocabulary to become familiar with: Tags and Elements

Tag = These are contained between a less than " < "and greater than " > " symbols. There is usually an open and close. The close tag has a forward slash / for example: < p> Element < /p> If a tag does not need a close it is called a void tag.
Element = Everything inside the opening and closing tags
Attribute = A property of an element. For example the anchor tag has the property href which you assign a link to.
